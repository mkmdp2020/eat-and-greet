# Let's Eat & Greet — Content Editing Guide

## How to edit your website content

1. Go to **letseatandgreet.com/admin**
2. Log in with GitHub
3. Click **All Site Content**
4. Edit any field
5. Click **Save** (top right)
6. Wait ~30 seconds → changes are live on letseatandgreet.com

---

## Business Info

These appear in the contact section, footer, and nav.

| Field | What it changes |
|---|---|
| Business Name | Name in nav and footer |
| Tagline | Short line under the name |
| Short Description | Text in the hero section |
| Logo | Upload your logo image (PNG/SVG recommended) |
| Email | Contact email shown on site |
| Phone / WhatsApp | Contact number (leave blank to hide) |
| Registered Address | Address shown in contact section |
| Service Area | e.g. Greater London & surrounding areas |
| Min Guests | Minimum number of guests you accept |
| Max Guests | Maximum number of guests you cater for |
| Opening Hours | When you accept enquiries |

## Hero Section

The big banner at the top of the page.

| Field | What it changes |
|---|---|
| Badge Text | Small pill text e.g. "Housewarming Specialists · Hayes" |
| Main Heading | Large headline text |
| Sub Text | Description paragraph below heading |

## About Section

The "Who We Are" section.

| Field | What it changes |
|---|---|
| Heading | Section title |
| Body Text | Main paragraph about the business |
| Feature Items | The 4 bullet points with icons |

**To edit a feature item:**
- Click the item to expand
- Change the Icon (use any emoji e.g. 🏠 🌶️ 👥)
- Change the Title (short bold label)
- Change the Detail (one sentence description)

---

## Menu

### South Indian Spotlight Dishes
The chip tags shown in the dark banner.
- Click **+** to add a new dish name
- Click the bin icon to remove one
- These are just labels — no description needed

### Menu Items
The cards shown in the menu grid.

**To add a new dish:**
1. Scroll to Menu Items → click **+**
2. Fill in:
   - **Dish Name** — e.g. "Pesarattu"
   - **Description** — one or two sentences
   - **Category / Tags** — e.g. "Starter · South Indian"
   - **Vegetarian?** — toggle on (green dot) or off (red dot)
3. Click Save

**To remove a dish:**
- Click the bin icon next to the item

---

## Packages

The three pricing/package cards.

| Field | What it changes |
|---|---|
| Package Name | Bold title of the card |
| Icon | Emoji shown above the name |
| Guest Range | e.g. "20 – 50 guests" |
| Description | Short sentence about who it suits |
| Highlight as Most Popular? | Adds the orange "Most Booked" badge |
| Included Features | Bullet list of what's included |

**To add a feature bullet:**
- Click **+** inside the package → type the feature text

---

## Testimonials

Client reviews shown in the dark section.

**To add a testimonial:**
1. Click **+** in the Testimonials section
2. Fill in:
   - **Client Name** — full name
   - **Event Type & Size** — e.g. "Housewarming — 45 guests"
   - **Quote** — what they said (no quotation marks needed)
   - **Initial** — single letter for the avatar e.g. "P"
   - **Avatar Colour** — pick a colour for the avatar circle

---

## Contact Form — Event Types

The dropdown options in the contact form.

- Click **+** to add a new event type
- Click the bin icon to remove one
- Order them however you like

---

## Footer

The footer pulls automatically from **Business Info**:
- Company name → from **Registered Company Name**
- Address → from **Registered Address**
- Email → from **Email**
- Copyright year → updates automatically each year

To change footer links or social media icons, ask your developer
(these are hardcoded in the HTML for now).

---

## Logo

1. In the CMS → click the **Media** icon (top left, looks like a photo)
2. Upload your logo file (PNG or SVG, ideally transparent background)
3. Name it `logo.png`
4. Go back to **All Site Content** → **Business Info** → **Logo**
5. Select the uploaded file
6. Save → logo appears in nav and footer automatically

---

## Tips

- **Changes take ~30 seconds** to go live after saving
- **Undo a change** — go to github.com/mkmdp2020/eat-and-greet → find the file → click History → restore a previous version
- **Phone number blank** = the field shows "Coming soon" on the site
- **Vegetarian toggle** = green dot (veg) / red dot (non-veg) on menu cards
- **Max 150 guests** is set in Business Info → Max Guests
